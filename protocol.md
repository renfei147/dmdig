# 通信协议 v1

## 发出请求

```json
{
    "video_id": 123
}
```

## 通知进度

```json
{
    "ok": false,
    "status": "获取弹幕中",
    "progress": 0.5
}
```

## 返回数据

```json
{
    "ok": true,
    "video_length": 123,
    "slice_length": 10,
    "slices": [
        {
            "n": 12,
            "w": {"233": 5, "666": 5, ...},
            "e": [1, 2, 3, 4, 5]
        },
        {
            "n": 16,
            "w": {"233": 5, "666": 5, ...},
            "e": [1, 2, 3, 4, 5]
        },
		...
    ]
}
```
